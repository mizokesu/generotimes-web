# DXTimes Web（記事検索フロントエンド）

DXTimes（記事を配信するデジタルWebメディア）のサイト内検索フロントエンドのデモ実装です。
記事検索の関連度ランキング刷新とファセット絞り込みの新UIを React（Vite）で試作しています。

> 注: デモ用のフロントエンドです。記事データは `src/articles.js` のモックを使用します（実データではありません）。

## 開発

```bash
npm install
npm run dev
```

## 構成

- `src/App.jsx` — 検索結果ページ（検索バー＋カテゴリ/著者ファセット）
- `src/ArticleCard.jsx` — 検索結果に表示する記事カード
- `src/articles.js` — モック記事データ

## 関連

検索リニューアルの設計・運用方針は、社内ナレッジベース「Rovo Demo」スペースおよび Jira プロジェクト `RD` を参照してください。
